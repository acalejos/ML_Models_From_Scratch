# ML Models From Scratch

Python implementations of several machine learning / data mining models from scratch using the speed dating model set CSV files

## Models Included:

* Naive-Bayes
* Logistic Regression
* Linear Support Vector Machine (SVM)
* Decision Tree
* Random Forests
* K-Means
